# OIBSIP
💻 Java Projects Suite

A curated collection of Java-based mini projects designed to demonstrate core programming concepts, object-oriented design, and real-world application development. This repository is ideal for students and beginners aiming to strengthen their Java fundamentals through practical implementation.

📖 Overview

This repository includes multiple console-based and web-oriented Java projects covering essential domains such as examination systems, banking interfaces, gaming logic, and data management systems. Each project focuses on modular design, clean structure, and usability.

🚀 Projects

📝 Online Examination System

A simulation of an online exam platform with essential functionalities for managing user sessions and assessments.

Key Features:

Secure user authentication (Login/Logout)
Profile and password management
Multiple-choice question (MCQ) handling
Timer-based auto submission
Session management

📚 Digital Library Management System

A complete system for automating library operations with user-friendly interaction.

Key Features:

Book inventory management
Issue and return tracking
Fine calculation system
Advanced booking functionality
Report generation

Modules:

Admin Module: Full control over books and user records
User Module: Search, browse, issue, and return books

🎯 Number Guessing Game

A simple interactive game demonstrating logic building and user input handling.

Key Features:

Random number generation
Hint system (higher/lower feedback)
Continuous gameplay loop

Optional Enhancements:

Attempt limits
Score tracking
Multi-round gameplay

🧾 Online Reservation System

A centralized reservation system simulating ticket booking and cancellation processes.

Key Features:

User authentication system
Reservation form with journey details
Auto-filled train information
Ticket cancellation using PNR
Centralized data handling

🏧 ATM Interface

A console-based banking system that mimics ATM operations.

Key Features:

Secure login (User ID & PIN)
Transaction history tracking
Cash withdrawal and deposit
Fund transfer functionality
Session termination (Exit)

🛠️ Tech Stack

Language: Java (Core Java)
Concepts: Object-Oriented Programming (OOP)
Interface: Console-based UI
Database (Optional): MySQL / JDBC (for extended functionality)

⚙️ Getting Started

Prerequisites

Java JDK (8 or above)
Any IDE (IntelliJ IDEA / Eclipse / VS Code)
Installation & Execution
Bash
# Clone the repository
git clone https://github.com/your-username/java-projects.git

# Navigate to project directory
cd java-projects

# Compile and run (example)
javac Main.java
java Main
